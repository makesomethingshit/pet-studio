# Contributing Ideas

Pet Studio is still small enough that useful contributions can be visual, technical, or documentation-focused.

## Good First Issues

- Add more room props that match the existing soft sticker style.
- Add more pet idle or waiting state variations.
- Add more speech bubble templates and state phrases.
- Improve preview/contact sheet layout and labels.
- Add new room themes for sample projects.
- Improve first-room install docs.
- Add more validation messages for common asset mistakes.
- Make sample screenshots easier to regenerate.
- Add small docs for creating helper pets that do not overpower the main pet.
- Improve Windows launcher troubleshooting notes.

## Slightly Bigger Ideas

- Add a richer project state demo script.
- Add more example project rooms.
- Improve hook event mapping docs.
- Add a portable recording recipe for the README GIF.
- Add test coverage around widget rendering edge cases.

## Contribution Tone

Keep the project local-first, playful, and clear. A good change should make the tiny room easier to understand, easier to run, or easier to trust.
